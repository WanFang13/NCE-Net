# NCE-Net
Code for "Nearest Neighbor Classifier Embedded Network for Active Learning", AAAI 2021


## Environments

* Ubuntu 18.04 LTS
* NVIDIA RTX 2080Ti
* Python 3.6
* Pytorch 1.1.0
* Cuda 10.2


## Train and Test

    
To perform train, test and evaluate, you can simply run:
 
    ```bash
    python main.py
    ```
